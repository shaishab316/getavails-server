## Features

- User Management:
  - Sign up
  - Sign in
  - Reset Password
  - Profile Management
- Chat Functionality:
  - Real-time chat
  - File sharing
  - Message deletion
- OTP Service:
  - Generate OTP
  - Verify OTP
- Stripe Integration:
  - Payment processing
  - Payment refund
- Request Validation:
  - Input validation
  - Authentication and Authorization
